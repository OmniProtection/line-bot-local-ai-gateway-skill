# LINE Bot Local AI Gateway Skill v0.1.1-alpha

## Status

Developer alpha. Not stable. Not production ready.

Repository: `OmniProtection/line-bot-local-ai-gateway-skill`

## Summary

This pre-release improves the public GitHub launch surface for `LINE Bot Local AI Gateway Skill`. It makes the README easier to understand at first glance, adds a 3-minute demo path, includes an architecture flow image, and prepares the repository for first-contributor discovery.

## What Changed

- Reworked the README first screen around the public value proposition: local-first LINE AI bot starter template with LM Studio, SQLite memory, explicit web search, and safety gates.
- Added a `3-Minute Demo` section with zero-secret checks, local template copy steps, and expected `/health`, memory, search, and remote-LLM safety outcomes.
- Added `docs/images/demo-flow.png` to show the LINE -> Webhook -> Safety Gate -> Memory/Search/LLM Router -> LM Studio flow.
- Added `docs/github-star-launch-plan.md` with recommended GitHub topics, repository description, launch copy, and first-10-stars checklist.
- Prepared starter issue topics for newcomer-friendly documentation improvements.

## What This Is

A local-first starter template for building LINE AI bots with LM Studio / OpenAI-compatible local model servers, SQLite memory, explicit web-search commands, and safety checks to avoid leaking secrets.

## Known Limitations

- Signature gate is `STATIC_VERIFIED`, not runtime verified.
- No live LINE smoke evidence is included.
- No real runtime evidence is included.
- No backup / restore drill evidence is included.
- Not suitable for production use without operator validation.
- Developer alpha only.

## Security Notes

Do not commit `.env`, LINE Channel Secret, Channel Access Token, reply tokens, search API keys, runtime SQLite databases, logs, backups, live webhook evidence, private tunnel URLs, local absolute paths, or LINE Developers Console screenshots with secrets.

If a remote LLM endpoint is configured, LINE messages, memory context, and search evidence may leave the operator machine. Remote LLM endpoints remain unsafe by default and require manual approval.
