## 這是什麼 / What This Is

這是一個本地優先的 LINE AI Bot starter template，可接 LM Studio / OpenAI-compatible local model server，內建 SQLite memory、明確觸發的 WebSearch 指令、config-gated Auto SearchPlan 判斷，以及避免誤提交 secrets 的安全檢查。

A local-first starter template for building LINE AI bots with LM Studio / OpenAI-compatible local model servers, SQLite memory, explicit web-search commands, config-gated Auto SearchPlan routing, and safety checks to avoid leaking secrets.

## 已知限制 / Known Limitations

- Signature gate 是 `STATIC_VERIFIED`，不是 runtime verified。
- 不包含 live LINE smoke evidence。
- 不包含 real runtime evidence。
- 不包含 backup / restore drill evidence。
- 沒有 operator validation 時不適合 production use。
- Developer alpha only。

## 安全提醒 / Security Notes

不要 commit `.env`、LINE Channel Secret、Channel Access Token、reply token、search API key、runtime SQLite database、logs、backups、live webhook evidence、private tunnel URL、local absolute path，或含 secret 的 LINE Developers Console screenshots。

如果設定 remote LLM endpoint，LINE messages、memory context、search evidence 可能離開 operator machine。Remote LLM endpoint 預設仍視為 unsafe，需要 manual approval。
